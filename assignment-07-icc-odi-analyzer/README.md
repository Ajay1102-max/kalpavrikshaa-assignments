# ICC ODI Player Performance Analyzer

This program implements the assignment "ICC ODI Player Performance Analyzer" in C.
It initializes player and team data from a provided header `Players_data.h` and
provides an interactive menu for analysis.

## Files
- `Players_data.h` : Provided header (do not modify)
- `icc_analyzer.c` : Main program (this file)
- `testcases.txt`  : Example input script

## Build
```bash
gcc icc_analyzer.c -o icc_analyzer
